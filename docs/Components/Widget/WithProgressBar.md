# WithProgressBar

## Requirements

N\A

## Usage

```twig
    <twig:widget:withProgressBar
        title="Widget withProgressBar"
        value="999 Mds."
    />
    
    <twig:widget:withProgressBar
        title="Widget withProgressBar"
        value="999 Mds."
        progress="25"
    />
    
    <twig:widget:withProgressBar
        title="Widget withProgressBar"
        value="999 Mds."
        trending="100"
        trendingUnit="%"
        progress="75"
    />
```
